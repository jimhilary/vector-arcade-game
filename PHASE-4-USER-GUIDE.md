## ğŸ’¡ GAMEPLAY TIPS (How to Win)

### For Beginners

1. **Learn to Fly First**
   - Practice moving without shooting
   - Get comfortable with rotation
   - Learn how inertia works (you keep moving!)

2. **Destroy Big Ones First**
   - Large asteroids are easier to hit
   - But they split into 2 medium ones
   - Plan your shots!

3. **Use Screen Wrapping**
   - Go off the right = appear on left
   - Go off top = appear on bottom
   - Use this to escape!

4. **Don't Stay Still**
   - Always keep moving
   - Asteroids will hit you!
   - Thrust, rotate, repeat

### For Advanced Players

1. **Chain Destruction**
   - Split large â†’ 2 medium (20 pts)
   - Split medium â†’ 2 small (50 pts each = 100 pts)
   - Destroy small (100 pts each = 200 pts)
   - **Total: 420 points from one large asteroid!**

2. **Control the Center**
   - Stay near screen center
   - More room to escape
   - See all asteroids at once

3. **Shoot Ahead**
   - Asteroids move fast
   - Aim where they're going
   - Not where they are!

4. **Save Your Lives**
   - You get 3 lives
   - After hit = 2 seconds invulnerable (flickering)
   - Use this time to reposition!

## ğŸ› TROUBLESHOOTING (If Something Doesn't Work)

### "I Don't See the Welcome Screen!"

**Check:**
1. Did the page load completely?
2. Try refreshing (F5)
3. Check browser console (F12) for errors

### "Leaderboard Shows 'Loading...' Forever"

**Fix:**
1. Make sure plugin is **activated**:
   - Go to `http://wp-vector-game.test/wp-admin/plugins.php`
   - Find "Vector Game API"
   - Click "Activate"

2. Check database table exists:
   - Go to phpMyAdmin
   - Look for `wp_vector_game_scores` table
   - If missing, deactivate & reactivate plugin

### "My Score Didn't Save!"

**Check:**
1. Did you enter a name?
2. Check browser console for errors
3. Verify database connection:
   ```
   Open phpMyAdmin
   Select database
   Browse wp_vector_game_scores table
   See if your score is there
   ```

### "Level-Up Message Doesn't Show"

**Try:**
1. Make sure you reached 500+ points
2. Check if modal is hidden behind canvas
3. Refresh page and try again

### "Game is Too Fast/Slow"

**Note:** Game runs at your monitor's refresh rate (usually 60 FPS)
- If too fast: Lower-end computer might help (weird but true!)
- If too slow: Close other programs, upgrade browser
